# Numerical Methods and Scientific Computing Toolkit

A collection of MATLAB implementations developed to study and analyze fundamental numerical methods used in scientific computing and applied mathematics.
The repository covers key topics including:
* Floating-point arithmetic and numerical error analysis
* Finite-difference approximations
* Matrix conditioning and stability analysis
* Gaussian Elimination with pivoting strategies
* Iterative solvers (Jacobi, Gauss-Seidel, SOR)
* Root-finding algorithms (Bisection, Newton-Raphson, Secant, Fixed Point)
* Multivariate Newton methods
* Polynomial interpolation (Lagrange and Newton methods)
* Cubic spline interpolation

The objective of this project is to gain practical understanding of numerical stability, convergence behavior, approximation accuracy, and computational efficiency through hands-on implementation and experimentation.
